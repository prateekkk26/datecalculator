# Days Between Two Dates Calculator

A fast, SEO-friendly, static web page for calculating days between two dates.

## Features

- Calculate days between two dates
- Toggle to include/exclude weekends
- Toggle to include/exclude start date
- Handles leap years and edge cases
- Mobile-friendly design
- SEO optimized with proper title, H1, and content

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

The built static files will be in the `dist` directory, ready for deployment.
